# Book Reviews & Canon

## Core Theory & Mental Models

- **Books compress contested history.** Great authors argue with rivals across decades; your job is to extract mechanisms, not slogans.
- **Canon without application is décor.** A principle earns shelf space when it changes a decision you would make next quarter.
- **Contradiction is information.** When two reputable frameworks disagree, map the boundary conditions—they rarely collide in the same problem class.

## Key Patterns & Decision Frameworks

- **Mechanism extraction:** Summarize claims as causal diagrams—inputs, feedback loops, failure modes—before rating prose quality.
- **Transfer checklist:** For each insight, document context fit (team size, regulation, tech era) to avoid cargo-cult adoption.
- **Reading debt:** Track unfinished volumes honestly; partial reads with extracted decisions beat pristine spines.

## Learning Path & Maturity Model

1. **Foundational:** Structured notes—thesis, evidence quality, counterarguments, and one concrete experiment to validate.
2. **Intermediate:** Comparative maps across authors—when each lens shines and where each breaks.
3. **Advanced:** Curricula built from primary sources plus practitioner corroboration—not summaries alone.
4. **Principal:** Institutional canon—shortlists that align onboarding and architectural debates without ideology policing.
