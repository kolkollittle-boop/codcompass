## 📐 间距系统

### 基础单位：8px

```
Spacing Scale:
4px    - 紧凑间距
8px    - 标准小间距
12px   - 中小间距
16px   - 标准间距
24px   - 中大间距
32px   - 大间距
48px   - 超大间距
64px   - 英雄区间距
```

### 应用示例

```css
/* 卡片内边距 */
.card { padding: 24px; }

/* 章节间距 */
.section { margin-bottom: 64px; }

/* 元素间距 */
.stack > * + * { margin-top: 16px; }
```

## 🎬 动效设计

### 原则

```
• 克制：只在必要时使用动效
• 快速：200-300ms 完成
• 自然：ease-out 缓动曲线
• 一致：全网站统一
```

### 常用动效

```css
/* 淡入 */
.fade-in {
  animation: fadeIn 0.3s ease-out;
}
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

/* 悬停 */
.hover-lift {
  transition: transform 0.2s ease-out;
}
.hover-lift:hover {
  transform: translateY(-2px);
}

/* 加载 */
.spinner {
  animation: spin 1s linear infinite;
}
@keyframes spin {
  to { transform: rotate(360deg); }
}
```

## 📱 响应式设计

### 断点

```css
/* 移动优先 */
--breakpoint-sm: 640px;   /* 大手机 */
--breakpoint-md: 768px;   /* 平板 */
--breakpoint-lg: 1024px;  /* 小屏电脑 */
--breakpoint-xl: 1280px;  /* 大屏电脑 */
```

### 适配规则

| 设备 | 宽度 | 布局 |
|------|------|------|
| **手机** | < 640px | 单列，全宽 |
| **平板** | 640-1024px | 双列 |
| **桌面** | > 1024px | 三列，侧边栏 |

## 🖼️ 图片风格

### 插图

```
• 使用简洁的线性插图
• 颜色与主色调一致
• 避免过于卡通化
• 参考：unDraw, ManyPixels
```

### 代码截图

```
• 使用统一的代码高亮主题
• 添加窗口装饰（红黄绿按钮）
• 圆角 + 阴影
• 最大宽度限制
```

### 图标

```
• 使用 Lucide / Heroicons
• 线条粗细一致 (1.5px)
• 颜色跟随上下文
• 尺寸统一 (20x20, 24x24)
```

## ✅ 设计检查清单

发布前确认：

```
□ 配色符合设计系统
□ 字体层级清晰
□ 间距统一（8px 基准）
□ 按钮状态完整（default/hover/active/disabled）
□ 响应式适配正确
□ 动效流畅不卡顿
□ 对比度符合 WCAG 2.1 AA
□ 深色模式兼容（可选）
```
