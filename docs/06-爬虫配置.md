# Codcompass Knowledge Base - 爬虫系统

## OpenClaw 爬虫配置

通过 OpenClaw cron jobs 定时采集内容，自动存入数据库。

### 数据源优先级

| 优先级 | 数据源 | 频率 | 方式 |
|--------|--------|------|------|
| 🔴 P0 | GitHub Trending | 每 2h | HTML 解析 |
| 🔴 P0 | Hacker News | 每 2h | Firebase API |
| 🔴 P0 | Reddit r/programming | 每 2h | JSON API |
| 🟡 P1 | Dev.to | 每 6h | 官方 REST API |
| 🟡 P1 | Hashnode | 每 6h | GraphQL API |

### 内容处理流程

```
抓取 → 去重 → AI 多源整合 → 改写 → 质量评分 → 存入 Supabase
```

## OpenClaw Cron Jobs 配置

### P0 爬虫（每 2 小时）

```
schedule: every 2h
payload: "从 GitHub Trending, Hacker News, Reddit r/programming 采集最新技术内容"
```

### P1 爬虫（每 6 小时）

```
schedule: every 6h
payload: "从 Dev.to, Hashnode 采集最新技术文章"
```

### 内容处理（每 10 分钟）

```
schedule: every 10min
payload: "处理待审核的文章：去重、AI 改写、质量评分"
```

## 质量标准

- 每篇文章必须整合 3-5 个不同来源
- 添加原创实战案例和代码示例
- AI 评分 ≥ 60 才能进入审核队列
- 发布前人工审核（MVP 阶段）
