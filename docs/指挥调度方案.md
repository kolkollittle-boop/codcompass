## 产品定位

### 目标市场

与 Aporia 项目一致，主打以下市场：
- **欧美** (英语、德语、法语、西班牙语)
- **新加坡** (英语、简体中文)
- **日本** (日语)
- **迪拜/中东** (英语、阿拉伯语)

### 多语言支持策略（v1.1 修订 — 先做英语+中文）

| 阶段 | 时间 | 语言 | 优先级 |
|------|------|------|--------|
| MVP | Week 1-2 | **仅英语**（后续加简体中文） | 🔴 P0 |
| 增长 | Week 5-12 | 简体中文、德语、法语、西班牙语 | 🟡 P1 |
| 成熟 | Week 13+ | 日语、繁体中文、阿拉伯语 | 🟢 P2 |

**技术方案**: next-intl + AI 自动翻译 (Claude Sonnet 4)

### 核心板块：AI Coding 工具学习知识库

按工具难度和收费层级划分：

| 层级 | 价格 | 包含工具 | 难度 | 目标人群 |
|------|------|----------|------|----------|
| 入门版 | $9.99/月 | Cursor, Cline | ★★☆ | 初学者，快速上手 |
| 专业版 | $29/月 | Claude Code, OpenCode | ★★★ | 进阶开发者 |
| 旗舰版 | $49/月 | Codex, OpenClaw + 自主创业 | ★★★★ | 高级玩家，创业者 |

### 收费模式：订阅制（MVP 先做这个）

> 💡 **MVP 阶段先只做订阅制**，验证后再考虑买断制。

#### 订阅制 (主体收入)
| 层级 | 价格 | 权益 |
|------|------|------|
| Builder 月订阅 | **$9.99/月** | 无限阅读 + 代码下载 + 每周通讯 |
| Pro 月订阅 | $29/月 | Builder 全部 + 源码仓库 + 视频教程 |
| Enterprise 月订阅 | $49/月 | Pro 全部 + 1v1 咨询 + 定制内容 |

**年订阅选项（打折促年付）：**
- Builder: $79/年（省 33%）
- Pro: $249/年（省 28%）

#### 买断制（Phase 3 再考虑）
| 产品 | 价格 | 内容 |
|------|------|------|
| 单工具入门指南 | $9.99 | Cursor/Cline 快速上手 PDF |
| 专项实战课程 | $19.99 | 完整项目 + 代码模板 |
| 配置模板包 | $4.99 | 即用型配置文件合集 |

#### 混合策略
1. **免费引流**: 每篇知识库文章提供前 30% 免费预览
2. **订阅解锁**: 付费订阅解锁全部文章 + 新内容
3. **买断补充**: 特定深度内容支持单独买断，永久访问
4. **捆绑优惠**: 买断 3 个产品 = 赠送 1 个月订阅

### 支付平台
- **主平台**: Lemon Squeezy (个人可用，处理全球税务)
- **备选**: Paddle (MoR 模式，功能类似)
- **订阅管理**: 支持升级/降级/取消，自动 webhook 同步

## 主流学习平台参考分析

参考 Udemy/Pluralsight/Frontend Masters 等主流平台，提炼可复用模式：

### 定价策略参考

| 平台 | 模式 | 价格区间 | 可复用点 |
|------|------|----------|----------|
| Udemy | 单课买断 | $9.99-$19.99 (打折) | 锚定效应：展示原价 vs 现价 |
| Pluralsight | 月订阅 | $29/月 | 阶梯定价：3 层，中间最受欢迎 |
| Frontend Masters | 年订阅 | $39/月 或 $399/年 | 年付折扣，降低决策成本 |
| Brilliant | 互动订阅 | $24.99/月 | 游戏化学习，进度追踪 |
| Codecademy | 免费增值 | 免费 + Pro $19.99/月 | 免费内容引流，付费解锁高级 |
| Gumroad | 混合模式 | 创作者自定 | 简单支付流程，创作者友好 |

### 转化优化策略

1. **社会证明**: 用户评价、学员数量、成功案例
2. **紧迫感**: 限时折扣、早鸟价格、倒计时
3. **退款保证**: 30 天无理由退款，降低决策风险
4. **免费试用**: 提供部分免费章节，引导购买
5. **捆绑销售**: 买旗舰版送入门版+专业版

### 内容结构参考

1. **路径化**: 从入门到精通的学习路径，每阶段有明确目标
2. **项目驱动**: 学完就能用的实战项目，不是纯理论
3. **模板下载**: 可直接复用的代码/配置模板，节省时间
4. **持续更新**: 新工具/新版本及时更新，保持价值
5. **社区互动**: 问答、讨论、分享，增加粘性

## 五阶段执行计划（v1.1 修订 — 加速版）

### 阶段一：项目初始化 + 基础网站（Week 1 前半段）

**目标:** Next.js 项目骨架 + 首页 + 知识库页面

| 步骤 | 操作 | 执行者 |
|------|------|--------|
| 1.1 | 创建 Next.js 项目骨架 + Supabase 配置 | Claude Code |
| 1.2 | 编写首页 (产品展示 + CTA) | Claude Code |
| 1.3 | 编写知识库页面 (文章列表 + 详情) | Claude Code |

**Claude Code 蜂群模式:**
```bash
claude --mode swarm -p "创建 cyberpunkweb Next.js 项目，集成 Supabase + NextAuth，简约专业风格"
```

### 阶段二：爬虫 + 内容生产 + 付费墙（Week 1 后半段 + Week 2）

## 风险与应对（v1.1 修订）

| 风险 | 应对 |
|------|------|
| 版权投诉 | 多源整合+结构重组+引用标注，前 20 篇人工审核 |
| 内容质量不稳定 | MVP 阶段人工审核，Claude Sonnet 4 保证质量 |
| Lemon Squeezy 配置复杂 | 先用测试模式，生产模式后配 |
| 网络/代理问题 | 本机 Clash 代理，Claude Code 有重试机制 |
| 用户增长缓慢 | Twitter/Reddit 免费内容引流 + Product Hunt 发布 |
