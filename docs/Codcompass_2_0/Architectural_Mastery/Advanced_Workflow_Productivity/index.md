# Advanced Workflow & Productivity

## Core Theory & Mental Models

- **Cognitive load accounting**: tools and processes compete for attention—productivity systems reduce context switching, not add dashboards.
- **Deep work protection**: architecture of calendars, review gates, and async norms shapes throughput more than individual hustle.
- **Automation paradox**: automating chaotic processes encodes chaos faster—stabilize workflow before scaling tooling.
- **Team cognitive systems**: shared mental models, glossary discipline, and ritual design outperform heroic individuals.

## Learning Path & Maturity Stages

1. **Workflow mapping** — Value stream from idea to production with wait states visualized.
2. **Friction removal** — Templates, ownership clarity, and elimination of duplicate approvals.
3. **Augmented workflows** — Assistive tooling with explicit human checkpoints for high-risk edits.
4. **Continuous refinement** — Retrospectives tied to measurable lead time and quality outcomes.
