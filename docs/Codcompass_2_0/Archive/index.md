# Archive

## Core Theory & Mental Models

- **History is not recommendation**: archived material preserves context for *why* past choices existed—not as default guidance.
- **Versioning narrative**: supersession links should point forward to current canon.
- **Legal & knowledge retention**: archive satisfies obligations without polluting active navigation.

## Learning Path & Maturity Stages

1. **Archive policy** — Criteria for freeze, migration, or deletion; ownership of archival metadata.
2. **Indexing discipline** — Search visibility flags so archived docs do not compete with current paths.
3. **Periodic review** — Sunset or redact content whose risk outweighs historical value.
