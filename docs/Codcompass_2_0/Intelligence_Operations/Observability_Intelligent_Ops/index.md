# Observability & Intelligent Ops

## Core Theory & Mental Models

- **Feedback velocity**: operations mature when signals shorten the loop from hypothesis to validated change.
- **Context-rich telemetry**: metrics alone under-specify; traces and bounded logs carry causal structure—within privacy envelopes.
- **Automation doctrine**: not everything should auto-remediate; classify by blast radius and reversibility.
- **Human–machine teaming**: operators steer automation with objectives; systems propose, humans authorize irreversible acts.

## Learning Path & Maturity Stages

1. **Signal foundations** — SLIs aligned to user journeys, not infrastructure vanity charts.
2. **Correlation discipline** — Consistent dimensions, exemplars, and trace propagation across async boundaries.
3. **Closed-loop actions** — Runbooks codified, progressively automated with simulation and audit trails.
4. **Learning systems** — Post-incident pattern mining without blaming individuals; architecture absorbs lessons.
